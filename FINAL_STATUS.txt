🚀 GETTING STARTED

1. INSTALL DEPENDENCIES
   Backend:
     cd backend
     npm install
   
   Frontend:
     cd frontend
     npm install

2. CONFIGURE ENVIRONMENT
   Backend:
     cp .env.example .env
     # Edit with your MongoDB URI and secrets
   
   Frontend:
     cp .env.example .env.local
     # Edit with your API URL

3. START SERVERS
   Backend:
     npm run dev
     # Runs on http://localhost:5000
   
   Frontend:
     npm run dev
     # Runs on http://localhost:3000

4. CREATE ADMIN ACCOUNT
   • Register normally on /register
   • In MongoDB, change role to "admin"
   • Login to access /admin

5. TEST EVERYTHING
   • See COMPLETE_TESTING_GUIDE.md
   • Run through all test cases
   • Verify all functionality
